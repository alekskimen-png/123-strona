# ⚽ Legendy Światowego Futbolu

Nowoczesna, interaktywna strona prezentująca **10 najlepszych piłkarzy w historii światowego futbolu**. Stworzona z pasji do piłki nożnej.

## ✨ Funkcje

- **10 legendarnych piłkarzy** – Pelé, Maradona, Messi, Ronaldo, Cruyff, Zidane, Ronaldinho, Beckenbauer, Ronaldo Nazário, Di Stéfano
- **Zdjęcia piłkarzy** z Wikipedii
- **Nowoczesny design** z ciemnym motywem i gradientami
- **Animacje** – floating elements, scroll reveal, hover effects
- **Responsywność** – mobile-first, hamburger menu
- **Tabela porównawcza** osiągnięć wszystkich legend
- **Ranking Top 10** z ocenami
- **Galeria cytatów** – kultowe wypowiedzi gwiazd futbolu

## 🛠 Technologie

- HTML5
- CSS3 (Flexbox, Grid, Animacje, Glassmorphism)
- JavaScript (Intersection Observer, płynne przewijanie)

## 📱 Podgląd

Strona zawiera:

1. **Hero** – efektowny nagłówek z animacją
2. **Legendy** – karty 10 najlepszych piłkarzy ze zdjęciami, statystykami i osiągnięciami
3. **Osiągnięcia** – tabela porównawcza (Złota Piłka, MŚ, LM, gole)
4. **Cytaty** – nieśmiertelne słowa futbolowych ikon
5. **Ranking** – subiektywny ranking Top 10 z ocenami
6. **Kontakt** – sekcja kontaktowa
