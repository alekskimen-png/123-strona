# ☕ Strona o Kawie

Prosta strona internetowa o kawie – rodzaje, przepisy i ciekawostki.

## Jak uruchomić lokalnie?

### Wystarczy otworzyć `index.html`
Po prostu kliknij dwukrotnie na plik `index.html` – otworzy się w przeglądarce.

### Albo użyj serwera (polecane)
Otwórz terminal w folderze i wpisz:

```
start index.html
```

lub uruchom prosty serwer Python:
```
python -m http.server 8000
```
Potem otwórz http://localhost:8000

## Struktura plików

- `index.html` – strona główna z GUI (karty nawigacyjne)
- `rodzaje.html` – rodzaje kawy
- `przepisy.html` – przepisy kawowe
- `ciekawostki.html` – ciekawostki
- `style.css` – stylizacja (ciemny motyw kawowy)
- `README.md` – niniejsza instrukcja
